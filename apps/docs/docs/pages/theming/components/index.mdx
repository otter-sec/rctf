# Components [UI component library overview]

The rCTF UI layer is built on [shadcn-svelte](https://shadcn-svelte.com/), a port of shadcn/ui for Svelte 5. Components live in `apps/web/src/lib/components/ui/` and use [tailwind-variants](https://www.tailwind-variants.org/) for variant-based styling. Application-specific components (Navigation, Markdown, ThemeToggle) are in the parent `components/` directory. Icons come from [Tabler Icons](https://tabler.io/icons) via `@iconify-svelte`.

The following components are available in rCTF:

| Category | Components |
|----------|------------|
| **Primitives** | `Button`, `Badge`, `Input`, `Textarea`, `Checkbox`, `Label`, `Separator`, `Progress`, `Spinner`, `Skeleton` |
| **Layout** | `Card`, `Section`, `ScrollArea`, `Resizable`, `Sidebar` |
| **Overlays** | `Dialog`, `Drawer`, `Sheet`, `Tooltip`, `Popover`, `DropdownMenu` |
| **Data** | `Table`, `Accordion`, `Tabs`, `VirtualList`\*, `Chart` |
| **Forms** | `Field`, `InputGroup`, `Select`, `TagInput`\*, `SchemaForm`\* |
| **Navigation** | `Command`, `Pagination`, `ButtonGroup` |
| **App** | `Navigation`\*, `NavigationMobile`\*, `ThemeToggle`\*, `Markdown`\*, `FlagPicker`\*, `EmptyState`\*, `SearchInput`\* |

\* These components are custom to rCTF and are not part of shadcn-svelte.

## Adding components

When adding a new shadcn-svelte component to rCTF, you need to install it, register it in the barrel file, and convert its styling to match rCTF's design system. This section walks through the full process using `Button` as an example.

:::steps

### Install the component

Use the shadcn-svelte CLI to scaffold the component into your project:

```bash [Terminal]
bunx shadcn-svelte@latest add button
```

This creates the component files in `apps/web/src/lib/components/ui/button/`.

### Register in the barrel file

Add the component to the barrel export in `apps/web/src/lib/components/index.ts`:

```ts [lib/components/index.ts]
export { Badge, badgeVariants, type BadgeVariant } from './ui/badge'
export { default as EmptyState } from './empty-state.svelte'
export { // [!code ++]
  Button, // [!code ++]
  buttonVariants, // [!code ++]
  type ButtonProps, // [!code ++]
  type ButtonSize, // [!code ++]
  type ButtonVariant, // [!code ++]
} from './ui/button' // [!code ++]
export { Checkbox } from './ui/checkbox'
export { Input } from './ui/input'
export { Label } from './ui/label'
```

### Remove shadows

rCTF uses a flat design language. Remove all shadow utilities from the component:

```ts [button.svelte]
export const buttonVariants = tv({
  base: "... shadow-xs", // [!code --]
  base: "...", // [!code ++]
  variants: {
    variant: {
      default: "bg-primary text-primary-foreground hover:bg-primary/90 shadow-xs", // [!code --]
      default: "bg-primary text-primary-foreground hover:bg-primary/90", // [!code ++]
      destructive: "bg-destructive hover:bg-destructive/90 ... shadow-xs", // [!code --]
      destructive: "bg-destructive hover:bg-destructive/90 ...", // [!code ++]
      outline: "bg-background hover:bg-accent ... shadow-xs", // [!code --]
      outline: "bg-background hover:bg-accent ...", // [!code ++]
      secondary: "bg-secondary text-secondary-foreground hover:bg-secondary/80 shadow-xs", // [!code --]
      secondary: "bg-secondary text-secondary-foreground hover:bg-secondary/80", // [!code ++]
    },
  },
})
```

### Normalize font weights

Convert all heavy font weights to `font-medium` for visual consistency:

| Original | Replace with |
|----------|--------------|
| `font-bold` | `font-medium` |
| `font-semibold` | `font-medium` |
| `font-black` | `font-medium` |
| `font-extrabold` | `font-medium` |

### Convert color tokens

Replace shadcn's color tokens with rCTF's [layered color system](/theming/colors). The mapping depends on context—use these as guidelines rather than strict rules:

| shadcn token | rCTF equivalent | Light | Dark | Context |
|--------------|-----------------|-------|------|---------|
| `bg-primary` | `bg-background-accent` | <Swatch color="background-accent" /> | <Swatch color="background-accent-dark" mode="dark" /> | Accent buttons, badges |
| `text-primary-foreground` | `text-foreground-accent` | <Swatch color="foreground-accent" /> | <Swatch color="foreground-accent-dark" mode="dark" /> | Text on accent backgrounds |
| `text-primary` | `text-foreground-accent` | <Swatch color="foreground-accent" /> | <Swatch color="foreground-accent-dark" mode="dark" /> | Accent-colored text |
| `bg-secondary` | `bg-background-l4` | <Swatch color="background-l4" /> | <Swatch color="background-l4-dark" mode="dark" /> | Secondary buttons, badges |
| `text-secondary-foreground` | `text-foreground-l1` | <Swatch color="foreground-l1" /> | <Swatch color="foreground-l1-dark" mode="dark" /> | Text on secondary backgrounds |
| `bg-muted` | `bg-background-l2` | <Swatch color="background-l2" /> | <Swatch color="background-l2-dark" mode="dark" /> | Subtle backgrounds |
| `text-muted-foreground` | `text-foreground-l3` | <Swatch color="foreground-l3" /> | <Swatch color="foreground-l3-dark" mode="dark" /> | Placeholders, hints |
| `bg-accent` | `bg-background-l2` | <Swatch color="background-l2" /> | <Swatch color="background-l2-dark" mode="dark" /> | Hover states, highlights |
| `text-accent-foreground` | `text-foreground-l0` | <Swatch color="foreground-l0" /> | <Swatch color="foreground-l0-dark" mode="dark" /> | Text on hover backgrounds |
| `bg-popover` | `bg-background-l1` | <Swatch color="background-l1" /> | <Swatch color="background-l1-dark" mode="dark" /> | Dropdowns, popovers |
| `text-popover-foreground` | `text-foreground-l0` | <Swatch color="foreground-l0" /> | <Swatch color="foreground-l0-dark" mode="dark" /> | Text in overlays |
| `bg-card` | `bg-background-l1` | <Swatch color="background-l1" /> | <Swatch color="background-l1-dark" mode="dark" /> | Card surfaces |
| `text-card-foreground` | `text-foreground-l0` | <Swatch color="foreground-l0" /> | <Swatch color="foreground-l0-dark" mode="dark" /> | Card text |
| `bg-background` | `bg-background-l0` | <Swatch color="background-l0" /> | <Swatch color="background-l0-dark" mode="dark" /> | Page background |
| `text-foreground` | `text-foreground-l0` | <Swatch color="foreground-l0" /> | <Swatch color="foreground-l0-dark" mode="dark" /> | Primary body text |
| `bg-destructive` | `bg-background-destructive` | <Swatch color="background-destructive" /> | <Swatch color="background-destructive-dark" mode="dark" /> | Error states |
| `text-destructive` | `text-foreground-destructive` | <Swatch color="foreground-destructive" /> | <Swatch color="foreground-destructive-dark" mode="dark" /> | Error text |
| `bg-input` | `bg-background-l4` | <Swatch color="background-l4" /> | <Swatch color="background-l4-dark" mode="dark" /> | Form inputs |
| `hover:bg-primary/90` | `hover:bg-background-accent-hover` | <Swatch color="background-accent-hover" /> | <Swatch color="background-accent-hover-dark" mode="dark" /> | Accent hover states |
| `hover:bg-secondary/80` | `hover:bg-background-l5` | <Swatch color="background-l5" /> | <Swatch color="background-l5-dark" mode="dark" /> | Secondary hover states |
| `hover:bg-accent` | `hover:bg-background-l2` | <Swatch color="background-l2" /> | <Swatch color="background-l2-dark" mode="dark" /> | Generic hover states |

Here's an example converting the `Badge` component variants:

```ts [badge.svelte]
variants: {
  variant: {
    default: "bg-primary text-primary-foreground [a&]:hover:bg-primary/90 border-transparent", // [!code --]
    default: "bg-background-accent text-foreground-accent [a&]:hover:bg-background-accent/90 border-transparent", // [!code ++]

    secondary: "bg-secondary text-secondary-foreground [a&]:hover:bg-secondary/90 border-transparent", // [!code --]
    secondary: "bg-background-l4 text-foreground-l1 [a&]:hover:bg-background-l5 border-transparent", // [!code ++]

    destructive: "bg-destructive [a&]:hover:bg-destructive/90 ... dark:bg-destructive/70 border-transparent text-white", // [!code --]
    destructive: "bg-background-destructive text-foreground-destructive [a&]:hover:bg-background-destructive/90 ... border-transparent", // [!code ++]

    outline: "text-foreground [a&]:hover:bg-accent [a&]:hover:text-accent-foreground", // [!code --]
    outline: "text-foreground-l1 [a&]:hover:bg-background-l2", // [!code ++]
  },
},
```

### Remove dark mode overrides

rCTF's color system handles dark mode automatically via CSS custom properties. Remove all `dark:` prefixed utilities:

```ts [button.svelte]
"dark:aria-invalid:ring-destructive/40 aria-invalid:border-destructive" // [!code --]
"aria-invalid:border-foreground-destructive" // [!code ++]

"dark:focus-visible:ring-destructive/40 dark:bg-destructive/60" // [!code --]
// (removed entirely) // [!code ++]

"dark:bg-input/30 dark:border-input dark:hover:bg-input/50" // [!code --]
// (removed entirely) // [!code ++]

"dark:hover:bg-accent/50" // [!code --]
// (removed entirely) // [!code ++]
```

### Remove transitions (optional)

rCTF components generally don't use transition utilities for a snappier feel. Remove if present:

```ts [button.svelte]
base: "... transition-all outline-none ...", // [!code --]
base: "... outline-none ...", // [!code ++]
```

## Full example

Here's the complete before and after for the Button component:

:::code-group

```svelte [Before (shadcn-svelte)]
<script lang="ts" module>
  import { cn, type WithElementRef } from "$lib/utils.js";
  import type { HTMLAnchorAttributes, HTMLButtonAttributes } from "svelte/elements";
  import { type VariantProps, tv } from "tailwind-variants";

  export const buttonVariants = tv({
    base: "focus-visible:border-ring focus-visible:ring-ring/50 aria-invalid:ring-destructive/20 dark:aria-invalid:ring-destructive/40 aria-invalid:border-destructive inline-flex shrink-0 items-center justify-center gap-2 rounded-md text-sm font-medium whitespace-nowrap transition-all outline-none focus-visible:ring-[3px] disabled:pointer-events-none disabled:opacity-50 aria-disabled:pointer-events-none aria-disabled:opacity-50 [&_svg]:pointer-events-none [&_svg]:shrink-0 [&_svg:not([class*='size-'])]:size-4", // [!code focus]
    variants: {
      variant: {
        default: "bg-primary text-primary-foreground hover:bg-primary/90 shadow-xs", // [!code focus]
        destructive:
          "bg-destructive hover:bg-destructive/90 focus-visible:ring-destructive/20 dark:focus-visible:ring-destructive/40 dark:bg-destructive/60 text-white shadow-xs", // [!code focus]
        outline:
          "bg-background hover:bg-accent hover:text-accent-foreground dark:bg-input/30 dark:border-input dark:hover:bg-input/50 border shadow-xs", // [!code focus]
        secondary: "bg-secondary text-secondary-foreground hover:bg-secondary/80 shadow-xs", // [!code focus]
        ghost: "hover:bg-accent hover:text-accent-foreground dark:hover:bg-accent/50", // [!code focus]
        link: "text-primary underline-offset-4 hover:underline", // [!code focus]
      },
      size: {
        default: "h-9 px-4 py-2 has-[>svg]:px-3",
        sm: "h-8 gap-1.5 rounded-md px-3 has-[>svg]:px-2.5",
        lg: "h-10 rounded-md px-6 has-[>svg]:px-4",
        icon: "size-9",
        "icon-sm": "size-8",
        "icon-lg": "size-10",
      },
    },
    defaultVariants: {
      variant: "default",
      size: "default",
    },
  });

  export type ButtonVariant = VariantProps<typeof buttonVariants>["variant"];
  export type ButtonSize = VariantProps<typeof buttonVariants>["size"];
  export type ButtonProps = WithElementRef<HTMLButtonAttributes> &
    WithElementRef<HTMLAnchorAttributes> & {
      variant?: ButtonVariant;
      size?: ButtonSize;
    };
</script>

<script lang="ts">
  let {
    class: className,
    variant = "default",
    size = "default",
    ref = $bindable(null),
    href = undefined,
    type = "button",
    disabled,
    children,
    ...restProps
  }: ButtonProps = $props();
</script>

{#if href}
  <a
    bind:this={ref}
    data-slot="button"
    class={cn(buttonVariants({ variant, size }), className)}
    href={disabled ? undefined : href}
    aria-disabled={disabled}
    role={disabled ? "link" : undefined}
    tabindex={disabled ? -1 : undefined}
    {...restProps}
  >
    {@render children?.()}
  </a>
{:else}
  <button
    bind:this={ref}
    data-slot="button"
    class={cn(buttonVariants({ variant, size }), className)}
    {type}
    {disabled}
    {...restProps}
  >
    {@render children?.()}
  </button>
{/if}
```
