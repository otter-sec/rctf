## Alerts

Alerts let you highlight important information in challenge descriptions using a blockquote-based syntax. Each alert type renders with a distinct color and icon.

```md
> [!NOTE]
> Background context or supplementary details.

> [!TIP]
> Suggestions to help players approach the challenge.

> [!IMPORTANT]
> Critical information required to solve the challenge.

> [!WARNING]
> Potential pitfalls or common mistakes to avoid.

> [!CAUTION]
> Actions that may have unintended consequences.
```

This outputs as the following:

<div style={{ maxWidth: '28rem', margin: '1.25rem auto', borderRadius: '0.5rem', overflow: 'hidden' }}>
![Alerts](/static/alerts.png)
</div>

You can also use `[!CONNECTION]{:md}` to add a copyable connection string to the challenge description:

```md
> [!CONNECTION]
> https://example.com

> [!CONNECTION]
> nc example.com 1337
```

This outputs as the following:

<div style={{ maxWidth: '28rem', margin: '1.25rem auto', borderRadius: '0.5rem', overflow: 'hidden' }}>
![Connection](/static/connection-alert.png)
</div>

:::info
You are free to omit inline code backticks (`` `nc example.com 1337` ``) when using `[!CONNECTION]{:md}`, as the output will always use a monospace font.
:::
