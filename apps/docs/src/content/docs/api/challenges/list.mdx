Returns the visible challenge list. Hidden challenges are omitted for non-admin users, and challenges with a future `releaseTime` are omitted until their release timestamp.

Use V2 for new clients. V2 adds instancer metadata, admin-bot input schemas, and the `hasFlag` field. V1 keeps the original challenge shape.
