Removes an external-auth client. After this returns, `<route>POST /api/v2/ext-auth/token</route>` exchanges with that `clientId` always fail with `<response>400 badExtAuthRequest</response>`.

Access tokens that were already minted through this client stay valid. There is no per-app token registry, so per-client revocation is not possible. If you need to invalidate outstanding tokens, rotate the global `tokenKey` (this invalidates every token in the system).

Unknown client IDs return `<response>400 badExtAuthRequest</response>`.

<RequestBody def={DeleteExtAuthClientRouteV2} source="params" title="Path parameters" />

#### Response

A successful request returns `<response>200 goodAdminExtAuthClientDelete</response>` with no data.
