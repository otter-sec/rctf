All colors in rCTF are defined using [OKLCH](https://oklch.com). Colors are declared as CSS custom properties in `apps/web/src/app.css{:file}`, with `:root{:css}` containing light mode values and `[data-theme='dark']{:css}` providing dark mode overrides. The theme is toggled via the `data-theme` attribute on `<html>{:html}`, persisted to `localStorage`, and respects `prefers-color-scheme` on first load.

## Color reference

### Layered colors

The foundation of rCTF's color system is an altered version of [shadcn/ui](https://ui.shadcn.com/docs/theming/)'s system. We add nested lightness layers 0 through 5, where layer `l0` represents the base (page background or primary text). Higher numbers progressively add emphasis (for backgrounds) or reduce prominence (for foregrounds).

| Layer | Background (light) | Background (dark) | Foreground (light) | Foreground (dark) |
| --- | --- | --- | --- | --- |
| `l0` | <Swatch color="background-l0" label="oklch(98% 0 0)" /> | <Swatch color="background-l0-dark" label="oklch(15% 0 0)" mode="dark" /> | <Swatch color="foreground-l0" label="neutral-700" /> | <Swatch color="foreground-l0-dark" label="neutral-50" mode="dark" /> |
| `l1` | <Swatch color="background-l1" label="oklch(95% 0 0)" /> | <Swatch color="background-l1-dark" label="oklch(18.5% 0 0)" mode="dark" /> | <Swatch color="foreground-l1" label="neutral-600" /> | <Swatch color="foreground-l1-dark" label="neutral-200" mode="dark" /> |
| `l2` | <Swatch color="background-l2" label="oklch(93% 0 0)" /> | <Swatch color="background-l2-dark" label="oklch(22% 0 0)" mode="dark" /> | <Swatch color="foreground-l2" label="neutral-500" /> | <Swatch color="foreground-l2-dark" label="neutral-300" mode="dark" /> |
| `l3` | <Swatch color="background-l3" label="oklch(91% 0 0)" /> | <Swatch color="background-l3-dark" label="oklch(26% 0 0)" mode="dark" /> | <Swatch color="foreground-l3" label="oklch(60% 0 0)" /> | <Swatch color="foreground-l3-dark" label="neutral-400" mode="dark" /> |

### Semantic colors

Semantic colors are used to convey meaning.

| Color | Light | Dark | Usage |
| --- | --- | --- | --- |
| `accent` | <Swatch color="background-accent" label="sky-200 @ 50%" /> | <Swatch color="background-accent-dark" label="sky-800 @ 30%" mode="dark" /> | Primary actions, buttons, links |
| `destructive` | <Swatch color="background-destructive" label="red-400 @ 30%" /> | <Swatch color="background-destructive-dark" label="red-950 @ 50%" mode="dark" /> | Error states, delete actions |
| `success` | <Swatch color="background-success" label="emerald-500 @ 10%" /> | <Swatch color="background-success-dark" label="emerald-900 @ 10%" mode="dark" /> | Success states, solved challenges |

### Scoreboard colors

Leaderboard colors are reserved for visualizing rankings on scoreboards and podiums, as well as for highlighting first, second, and third bloods. Gold, silver, and bronze semantically denote the top three placements (see [Wikipedia's "Hierarchy of precious substances"](https://en.wikipedia.org/wiki/Hierarchy_of_precious_substances)), while "self" highlights the current user's row. The "nth" color is applied to all other participants outside the top three, and is just mappings of our layered color system.

| Color | Background (light) | Background (dark) | Foreground (light) | Foreground (dark) |
| --- | --- | --- | --- | --- |
| `gold` | <Swatch color="background-gold" label="yellow-500 @ 15%" /> | <Swatch color="background-gold-dark" label="yellow-950 @ 20%" mode="dark" /> | <Swatch color="foreground-gold-l0" label="yellow-600" /> | <Swatch color="foreground-gold-l0-dark" label="yellow-400" mode="dark" /> |
| `silver` | <Swatch color="background-silver" label="slate-400 @ 20%" /> | <Swatch color="background-silver-dark" label="slate-700 @ 20%" mode="dark" /> | <Swatch color="foreground-silver-l0" label="slate-600" /> | <Swatch color="foreground-silver-l0-dark" label="slate-300" mode="dark" /> |
